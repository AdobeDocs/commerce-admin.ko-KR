## [!UICONTROL Configure Allowed File Types and Size]

| 필드 | [범위](../../getting-started/websites-stores-views.md#scope-settings) | 설명 |
|--- |--- |--- |
| [!UICONTROL Allowed File Extensions] | 글로벌 | 허용되는 파일 형식을 쉼표로 구분한 목록(예: `pdf,doc,docx,txt`). pdf, doc, docx, xls, xlsx, ppt, pptx, txt, csv, zip 등 허용되는 기본 파일 유형을 사용하려면 비워 둡니다. |
| [!UICONTROL Maximum File Size] | 글로벌 | 허용되는 최대 파일 크기(MB)(예: `20.0`). 기본 제한은 16MB입니다. 허용되는 최대 파일 크기는 31MB입니다. |
