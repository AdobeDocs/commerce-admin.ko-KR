# 콘텐츠 블록

_블록_&#x200B;은(는) 페이지의 어디에나 배치할 수 있는 모듈식 콘텐츠 단위입니다. 콘텐츠 블록을 _정적 블록_ 또는 _CMS 블록_&#x200B;이라고도 합니다. 텍스트, 이미지 및 포함된 비디오와 같은 고정 정보와 위젯에서 제공하거나 데이터베이스 또는 기타 소스에서 가져온 동적 정보를 표시하는 데 사용할 수 있습니다. 홈 페이지의 요소 대부분은 쉽게 관리할 수 있는 블록입니다.

코드를 작성하지 않고 사용자 지정 콘텐츠 블록을 만들어 페이지 레이아웃의 특정 위치에 나타나도록 지정할 수 있습니다. [위젯](widget-static-block.md) 도구를 사용하거나 XML로 [레이아웃 업데이트](layout-updates.md)를 작성하여 서버에 저장하여 블록을 배치할 수 있습니다. 레이아웃 업데이트 사용에 대한 자세한 내용은 [프론트엔드 개발자 안내서](https://developer.adobe.com/commerce/frontend-core/guide/layouts/)에서 _레이아웃_ 정보를 참조하십시오.
