![해결된 문제](../assets/fix.svg) 컨텐츠 편집으로만 제한되는 권한을 가진 관리자는 Page Builder를 사용하여 제품 위젯을 CMS 페이지에 추가할 때 더 이상 오류가 표시되지 않습니다. 또한 Page Builder는 위젯 설정 페이지에 정확한 제품 수를 표시합니다. 이전에는 제품 수를 검색할 때 Page Builder에 카탈로그 모듈에 대한 권한이 필요했고 이 오류를 표시했습니다. `A technical problem with the server created an error. ![해결된 문제](../assets/fix.svg) 이제 마우스를 사용하여 **표시할 텍스트** 값을 지정한 경우 이해할 수 있도록 해줍니다. <!--- AC-982-->

![해결된 문제](../assets/fix.svg) 슬라이더를 작은 열에 표시할 때 페이지 빌더 가 더 이상 링크 삽입 및 이미지 삽입 대화 상자의 크기를 조정하지 않습니다. <!--- AC-973-->

## Adobe Commerce 2.4.2용 1.6.0

![신규](../assets/new.svg) <!-- Issue 558 -->**신규 [!DNL Page Builder] 스타일링** - [!DNL Page Builder] 콘텐츠 스타일링 방법을 크게 개선했습니다. 이제 이러한 변경 사항으로 간단한 CSS 선택기를 만들어 기존 콘텐츠 유형 스타일을 재정의할 수 있습니다. 이러한 변경 사항으로 다음을 생성할 수 있습니다. [!DNL Page Builder] 모든 콘텐츠 유형에 대해 풍부한 응답성을 갖는 테마입니다.

![신규](../assets/new.svg) <!-- Issue 429 -->**이제 행 컨테이너 선택 사항** - 이제 콘텐츠를 만들 수 있습니다. [!DNL Page Builder] 행 컨테이너를 필요로 하지 않습니다. 이제 행 외에도 HTML, 블록, 동적 블록, 열 및 탭과 같은 콘텐츠 유형을 스테이지에서 바로 드래그하여 놓을 수 있습니다.

![신규](../assets/new.svg) <!-- Issue 636 -->**새 반응형 뷰포트 전환기** - 이제 다음을 미리 볼 수 있습니다. [!DNL Page Builder] 단계 위에 있는 새로운 관리 뷰포트 전환기 버튼을 사용하여 다양한 디바이스 너비 (중단점)의 콘텐츠를 표시합니다. 뷰포트 전환기는 서로 다른 장치를 사용하여 상점에 표시될 때 콘텐츠의 스타일이 지정되는 방식을 시뮬레이션합니다.

![신규](../assets/new.svg) <!-- Issue 637 -->**새 중단점별 양식 필드** - 이제 콘텐츠 유형 필드 값을 특정 뷰포트 중단점으로 설정할 수 있습니다. 필드 옆의 아이콘 표시기는 콘텐츠 유형 필드 값이 적용되는 뷰포트를 보여 줍니다.

![신규](../assets/new.svg) <!-- Issue 559 -->**사전 정의된 양식 필드 항목이 제거됨** - 이제 여백, 패딩 및 테두리 관련 속성(테두리, 테두리 폭 및 테두리 반경)에 대한 고급 양식 설정이 `default`를 사용하여 모듈의 CSS로 값을 정의할 수 있습니다.

![신규](../assets/new.svg) <!-- Issue 635, 557,  -->**스테이지 간격이 추가되었습니다.** - 이제 행 및 열은 스테이지에서 포함된 콘텐츠 유형에 대해 추가 간격을 허용하지만 상점 환경에서는 허용하지 않습니다. 추가 단계 간격을 사용하면 컨테이너 및 포함된 콘텐츠 유형 모두에 대한 마우스오버 옵션 메뉴에 더 쉽게 액세스할 수 있습니다.

![해결된 문제](../assets/fix.svg) <!-- MC-37348 -->**검토에 대한 자동 스크롤이 수정됨** - 상점에서의 제품 리뷰로의 자동 스크롤이 이제 수정되었습니다.

![해결된 문제](../assets/fix.svg) <!-- MC-36227 -->**잘린 콘텐츠가 수정됨** - 긴 카테고리 및 제품 설명이 더 이상 잘리지 않습니다.

![해결된 문제](../assets/fix.svg) <!-- MC-35402 -->**고정 전체 너비, 전체 도련 카테고리 콘텐츠** - 이제 카테고리 컨텐츠를 전체 너비 또는 전체 도련 레이아웃으로 표시할 수 있습니다.
